Text mining:

Here we will use the tidy text package with the unnest_tokens function to text mine our data.the unnest_tokens package takes texts and breaks it into individual words. In this way, there will be just one word per row in the dataset. this makes it possible for us to target the word âtobakâ. This helps us clean it and now our data isnât messy any more and now we have Tobak_tidy.

Tobak_tidy <- Tobak_1850_1885 %>% 
  unnest_tokens(word, fulltext_org)

To clean the data more we loaded in a stopword list which will sort out all the most common words such as âatâ, âdetâ etc. We got the stopwordlist from Max Odbjerg.
